This handler sets the log-file that is used for calls of handler log message.
The file parameter can be a POSIX file, an alias or a string (in POSIX or alias format)
Example:
enable logging to file "/var/tmp/mylogfile.log"
This handler logs a text to console or file.
The file can be specified by calling the enable logging to file handler.
The message is preceded by a date and time stamp and a log level.
The available log levels are info, alarm, fault and severe.
These levels are displayed in the log as INFO, WARNING, ERROR and SEVERE.
Example:
log message "aVariable=" & aVariable as severe
